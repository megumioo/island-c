GitHub 云同步配置

手动配置 GitHub PAT

island 使用 GitHub Personal Access Token (PAT) 安全地同步数据到您的私有 Gist。

访问 GitHub → Settings → Developer settings → Personal access tokens
点击 "Generate new token" → "Generate new token (classic)"
填写 Token 描述（如："island sync"）
勾选权限：gist（必需）
点击 "Generate token" 并复制生成的 token
Token 仅存储在您的浏览器中，用于安全的云同步

注意事项：

  • 确保 token 以 ghp_ 或 github_pat_ 开头
  • token 需要 gist 权限来创建/更新私有 Gist
  • 数据将加密存储在您的私有 Gist 中
  • 您可以随时在 GitHub 设置中撤销此 token
